# Proxy-for-telegram

Прокси скрипт для работы телеграм.

**Используемые адреса**

  nl13.postls.com:443
  nl5.postls.com:443
  nl6.postls.com:443
  uk8.postls.com:443
  uk19.postls.com:443

**Используемые адреса:**

	ufw deny from nl13.postls.com:443
	ufw deny from nl5.postls.com:443
	ufw deny from nl6.postls.com:443
	ufw deny from uk8.postls.com:443
	ufw deny from uk19.postls.com:443
	sudo ufw enable
